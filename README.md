# Language Translator GUI (Python)

A desktop language translation application built with **Python** and **Tkinter**.
The app allows users to translate text between multiple languages using the
**Deep Translator (Google Translate)** API.

## Features
- Translate between 10+ languages
- Dropdown language selection
- Multi-line text input and output
- Keyboard shortcuts (Enter to translate, Esc to quit)
- Input validation and error handling

## Technologies Used
- Python
- Tkinter
- deep-translator
